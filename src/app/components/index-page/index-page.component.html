Welcome to my FullStack Practice Project

I developed the Grocery Notes FullStack WebApp to deepen my understanding of both front-end and back-end web development.

Front-end Achievements:

  • Used Angular & Bootstrap to build a dynamic and user-friendly web interface.
  • Enhanced my HTML/CSS skills, focusing on clean and intuitive design.
  • Implemented JWT authentication for secure user sessions.

Back-end Achievements:

  • Hosted my .NET REST API ensuring 24/7 accessibility.
  • Managed PostgreSQL databases with Entity Framework, handling user data and grocery lists.

Deployment and Security:

  • Deployed on a DigitalOcean droplet for reliable uptime.
  • Set up Nginx as a reverse proxy to manage and secure traffic.
  • Configured DNS for my domain and ensured end-to-end security.
  • Gained hands-on experience with cloud platforms, especially AWS.

This project was a significant step in my web development journey. I'm eager to leverage this knowledge and take on even bigger challenges.
